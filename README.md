Cryptography Project
A C# application implementing classical and modern cryptographic algorithms for encrypting and decrypting text messages.
What it does
Provides implementations of multiple encryption algorithms, allowing users to encrypt plaintext messages and decrypt ciphertext using various classical and symmetric cryptographic techniques.
Algorithms Implemented

Caesar Cipher
Vigenère Cipher
Transposition Cipher
(Additional algorithms — see source code)

How it works

User selects an encryption algorithm
Enters the plaintext message and a key
The application encrypts the message and displays the ciphertext
Decryption is also supported using the same key

Use Case
Built as part of a cryptography and network security course to understand the foundations of encryption by implementing the algorithms from scratch.
